# Project Brief

## Project Name

Doctor Note MVP

## Vision

Build a lightweight web application for small clinics to manage doctors, patients, consultation notes, and patient history digitally.

## Problem Statement

Small clinics often use paper notes or spreadsheets, making patient history difficult to search and maintain.

## Target Users

- Clinic Admin
- Doctor
- Receptionist

## Core Features

- Authentication
- Role-based access
- Doctor management
- Patient management
- Patient profile
- Consultation notes
- Patient history
- Dashboard
- Search
- PDF export

## Nice-to-Have Features

- AI consultation summary
- Voice dictation
- Dark mode

## Out of Scope

- Billing
- Appointment scheduling
- Insurance
- Pharmacy
- Laboratory integration
- Mobile app
- Multi-clinic support

## Technology

- Next.js
- TypeScript
- Supabase
- PostgreSQL
- Tailwind CSS
- shadcn/ui
- Vercel

## Timeline

2 weeks

## Success Criteria

- User can log in.
- Admin can manage doctors.
- Receptionist can register patients.
- Doctor can create consultation notes.
- User can view patient history.
- User can export consultation PDF.
